# 개요
- 대구동신교회 청년부 '아포슬' 2026년도 전체여름수련회를 위한 웹앱

Re:本, Re:born, Re:bond


# 개발 환경
- 백엔드 호스팅: Vercel Functions(Vercel Pro)
- DB 서버: Vercel Storage
- 파일베이스: Vercel Blob
- Realtime: Supabase
- 언어: 프론트(Next.js), 백엔드(TypeScript)


# 결제

1. Claude Pro
- Pro
- $20 (약 2.7만 원)
- 최신 모델(3.5 Sonnet 등)
- 우선 접근 및 대화 횟수 증폭

2. Vercel Pro
- Pro
- $20 (약 2.7만 원)
- 상업적 이용 가능, 서버리스 함수 실행 시간 연장, 팀 협업 기능

3. Supabase
- Pro
- $25 (약 3.4만 원)
- 1,000명 동시 접속 및 실시간(Realtime) 채팅 안정성 보장

4. Vercel Storage(Vercel Pro에 포함, 추가 결제)
- Postgres
- $0 ~ $10+
- 기본 사용량 무료, 데이터 입출력(Compute/Storage) 양에 따라 추가 과금

5. Vercel Blob(Vercel Pro에 포함, 추가 결제)
- Blob
- $0 ~ $5+
- 기본 250MB 무료, 이후 저장 용량 및 다운로드 대역폭당 과금

# 개발 기능

## 마이프로필
> - 메인 헤더의 프로필 아이콘 클릭시 마이프로필 이동
> - 이름, 전화번호, 팀, 조, 숙소, 참석구분 표시

## 공지사항
> 1. 메인 페이지
> - admin에서 등록한 공지 표출
> - 공지 타이틀
> - 공지 시간(방금, 몇분전, 몇시간전, 어제 등)
> - 공지 내용
> - 최근 공지 3개까지만 표시(가장 최신 공지 강조)
> - 공지 클릭 시 모달로 전체 내용 확인
> - 시간 관련 공지는 초록색으로 강조
> - 긴급 공지는 적색으로 강조(사건사고 및 안전 관련)

> 2. 어드민 페이지
> - 공지 등록(타이틀, 내용, 분류)
> - 공지 등록시 분류 기준(일반, 시간, 긴급)
> - 일반은 보통의 공지
> - 시간은 시간 관련 공지(타임스케줄 안내 등)(초록색으로 강조)
> - 긴급은 긴급한 공지(빨강색으로 강조)
> - 공지 수정 및 삭제 기능
> - 등록한 공지 히스토리

## 타임테이블
>1. 메인 페이지
>- 현시각 진행중인 프로그램의 타이틀과 시간
>- 다음 진행 예정인 프로그램의 타이틀과 시간

>2. 타임테이블 페이지
>- Day1(6/5금), Day2(6/6토), Day3(6/7일) 별 스케줄
>- 현 디자인 유지

>3. 어드민 페이지
>- 타임테이블 관리
>- 등록(타이틀, 장소, 시간, 타입)
>- 타입(이동, 식사, 소그룹, 예배, 특별)
>- 수정 및 삭제 기능

## 기도제목
>1. 기도제목 페이지
>>* 40일 기도릴레이 체크포인트(수련회 시작 시 삭제 예정)
>
>>* 기도제목 올리기
>>- 기도제목 입력란
>>- 익명으로 올리기 체크
>>- "공유되며, 다른 참가자들이 함께 기도합니다"
>
>>* 기도제목 확인
>>- 나의 기도제목: 내가 올린 기도제목 리스트 확인, 수정 및 삭제, 익명 전환
>>- 우리 조 기도제목: 조원들이 올린 기도제목 리스트 확인
>>- 모두의 기도제목: 실명, 익명으로 올린 모든 기도제목 확인

>2. 어드민 페이지
>- 참가자별 기도제목 확인, 수정, 삭제, 익명 전환
>- 조별 기도제목 확인, 수정, 삭제
>- 전체 기도제목 확인, 수정, 삭제

## 참가자 관리
>1. 어드민 페이지
>- 참가자 등록(이름, 팀(초신자팀,기신자팀,1팀,2팀,3팀,4팀,5팀,6팀,웰컴팀,임원단), 전화번호, 참석구분(A형, B-1형, B-2형, C형, D형))
>- 참가자 관리(검색, 정보 수정, 삭제)
>- 조 편성(조 생성, 조별 조원 추가, 조원 삭제, 조장 선택)
>- 숙소 편성(숙소 생성, 숙소별 참가자 추가, 참가자 삭제)

## 내 조 확인 페이지
>- 메인의 빠른 접근 메뉴에 내 조 확인 카드 클릭으로 이동
>- 조장, 조원의 이름, 전화번호, 팀, 숙소 표시
